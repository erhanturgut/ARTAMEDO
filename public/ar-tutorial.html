🎮 AR Animasyon Rehberi

A-Frame ile kendi 3D animasyonlarınızı oluşturun

Temel Kavramlar

A-Frame, web tabanlı 3D ve VR/AR deneyimleri oluşturmak için kullanılan bir HTML çerçevesidir. Basit HTML etiketleri ile 3D nesneler ve animasyonlar oluşturabilirsiniz.

Koordinat Sistemi

A-Frame, sağ el koordinat sistemini kullanır:

  • X ekseni: Sağ (+) / Sol (-)
  • Y ekseni: Yukarı (+) / Aşağı (-)
  • Z ekseni: Önde (+) / Arkada (-)

İpucu

Marker üzerinde görünmesi için nesnelerin Y değerini 0'dan büyük yapın (örn: position="0 0.5 0").

Temel Şekiller (Primitives)

A-Frame'de kullanabileceğiniz hazır 3D şekiller:

📦 Kutu (Box)

<a-box position="0 0.5 0" color="#4CC3D9"></a-box>

width, height, depth ile boyutlandırılır

🔵 Küre (Sphere)

<a-sphere position="0 0.5 0" radius="0.5" color="#EF2D5E"></a-sphere>

radius ile boyutlandırılır

🛢️ Silindir (Cylinder)

<a-cylinder position="0 0.5 0" radius="0.3" height="1" color="#FFC65D"></a-cylinder>

radius ve height ile boyutlandırılır

🔺 Koni (Cone)

<a-cone position="0 0.5 0" radius-bottom="0.5" height="1" color="#7BC8A4"></a-cone>

radius-bottom ve height ile boyutlandırılır

🍩 Halka (Torus)

<a-torus position="0 0.5 0" radius="0.4" radius-tubular="0.1" color="#9B59B6"></a-torus>

radius ve radius-tubular ile boyutlandırılır

✏️ Metin (Text)

<a-text value="Merhaba" position="0 0.5 0" color="white"></a-text>

value, scale, align ile özelleştirilir

Animasyonlar

Nesnelere animasyon eklemek için animation özelliğini kullanın:

<a-box position="0 0.5 0" color="#4CC3D9"
  animation="property: rotation; to: 0 360 0; dur: 2000; loop: true"></a-box>

Animasyon Özellikleri

Özellik Açıklama Örnek
property Hangi özelliğin animasyonlu olacağı rotation, position, scale, material.color
from Başlangıç değeri (opsiyonel) 0 0 0
to Bitiş değeri 0 360 0
dur Süre (milisaniye) 2000 (2 saniye)
loop Tekrar true, false, 3 (sayı)
dir Yön normal, reverse, alternate
easing Geçiş efekti linear, easeInOut, easeOutElastic
delay Gecikme (ms) 500

Çoklu Animasyon

Bir nesneye birden fazla animasyon eklemek için animation__isim formatını kullanın:

<a-sphere position="0 1 0" color="#EF2D5E"
  animation="property: position; to: 0 2 0; dur: 1000; dir: alternate; loop: true"
  animation__rotate="property: rotation; to: 0 360 0; dur: 2000; loop: true"
  animation__color="property: material.color; to: #00ff88; dur: 1500; dir: alternate; loop: true"></a-sphere>

Örnek Animasyonlar

Dönen Küp 
<a-box position="0 0.5 0" color="#4CC3D9"
  animation="property: rotation; to: 360 360 0; dur: 3000; loop: true; easing: linear"></a-box>
Zıplayan Top 
<a-sphere position="0 0.5 0" radius="0.3" color="#EF2D5E"
  animation="property: position; from: 0 0.3 0; to: 0 1.5 0; dur: 800; dir: alternate; loop: true; easing: easeOutQuad"></a-sphere>
Yanıp Sönen Işık 
<a-sphere position="0 0.5 0" radius="0.3" color="#ffeb3b"
  animation="property: material.opacity; from: 0.3; to: 1; dur: 500; dir: alternate; loop: true">
  <a-light type="point" color="#ffeb3b"
    animation="property: intensity; from: 0; to: 2; dur: 500; dir: alternate; loop: true"></a-light>
</a-sphere>
Hareket Eden Partikül 
<a-sphere position="-1 0.2 0" radius="0.05" color="#00ff88"
  animation="property: position; from: -1 0.2 0; to: 1 0.2 0; dur: 2000; loop: true"></a-sphere>
Büyüyüp Küçülen Nesne 
<a-box position="0 0.5 0" color="#9B59B6"
  animation="property: scale; from: 0.5 0.5 0.5; to: 1 1 1; dur: 1000; dir: alternate; loop: true"></a-box>

Elektrik Devresi Örneği

Basit bir elektrik devresi animasyonu:

<!-- Pil -->
<a-cylinder position="-0.5 0.15 0" radius="0.08" height="0.2" color="#ff6b6b"></a-cylinder>
<a-text value="+" position="-0.5 0.3 0" scale="0.3 0.3 0.3" color="white" align="center"></a-text>

<!-- Kablo -->
<a-box position="0 0.1 0" width="1" height="0.02" depth="0.02" color="#00d4ff"></a-box>

<!-- Ampul -->
<a-sphere position="0.5 0.2 0" radius="0.1" color="#ffeb3b"
  animation="property: material.emissiveIntensity; from: 0; to: 1; dur: 800; dir: alternate; loop: true"></a-sphere>

<!-- Akım partikülleri -->
<a-sphere radius="0.02" color="#00ff88"
  animation="property: position; from: -0.5 0.1 0; to: 0.5 0.1 0; dur: 1500; loop: true"></a-sphere>

İpuçları

Boyut

Nesneleri çok büyük yapmayın. 0.1 - 1 birim arası en iyi görünür.

Konum

Nesneleri Y=0 seviyesinin üstüne yerleştirin (örn: Y=0.5), böylece marker'ın üzerinde görünürler.

Animasyon Hızı

Çok hızlı animasyonlar göz yorar. 1000-3000ms arası ideal.

Test

Animasyonlarınızı önce A-Frame Inspector'da test edin.
